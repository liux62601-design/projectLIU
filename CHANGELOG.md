# Changelog

All notable changes to TechTutorial Pro will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-22

### Added
- Initial project scaffolding and foundation files
- `.gitignore` for static web project
- `.nojekyll` to disable Jekyll processing on GitHub Pages
- `robots.txt` with sitemap reference
- `sitemap.xml` with core site pages
- `manifest.json` for PWA support
- MIT `LICENSE`
- `SECURITY.md` security policy
- `CONTRIBUTING.md` contribution guidelines
- `CODE_OF_CONDUCT.md` based on Contributor Covenant 2.0
- `CHANGELOG.md` for tracking project changes
